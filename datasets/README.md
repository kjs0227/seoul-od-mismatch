# Datasets

This directory contains:

* `HTS_sample.csv`: unprocessed sample records preserving the structure of the original HTS data;
* `LCS_sample.csv`: unprocessed sample records preserving the structure of the original hourly LCS data; and
* `Regression_input.csv`: TAZ-level spatial, socioeconomic, and survey attributes used in the regression analysis.

The HTS and LCS OD flows used in the gravity models are generated by the preprocessing notebooks.
The original HTS and LCS datasets are not redistributed. The sample files are provided only to illustrate the structure of the source data before preprocessing.
